Emacs: статический анализ Ruby-кода

Rubocop – это полезный инструмент для любого Ruby-разработчика. Rubocop осуществляет статический анализ Ruby-кода основываясь на Ruby Style Guide и помогает сделать твой код (или код твоих коллег) чуточку читабельнее (а значит и лучше).

Вот только есть один минус – ручной запуск Rubocop из терминала. Сам понимаешь, хочется интеграции этого чуда в среду разработки, так чтобы оно само. Ну что же, давай заинтегрируем.
